# Polkadot Bulletin SDK

SDKs for storing data on Polkadot Bulletin Chain.

| Language | Package | Status |
|----------|---------|--------|
| Rust | `bulletin-sdk-rust` | Alpha |
| TypeScript | `@bulletin/sdk` | Alpha |

Both SDKs provide automatic chunking, DAG-PB manifest generation (IPFS-compatible), CID calculation, and authorization management.

## Getting Started

- [Core Concepts](./concepts/README.md) - how storage and authorization work
- [Rust SDK](./rust/README.md) - native apps, ink! contracts, no_std/WASM
- [TypeScript SDK](./typescript/README.md) - Node.js and browser apps with PAPI
