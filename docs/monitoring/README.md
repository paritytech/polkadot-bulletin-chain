# Grafana dashboards

Grafana dashboard models for the Bulletin networks. Edit here, then re-import.

- `bulletin-summit-dashboard.json` — Summit
- `bulletin-paseo-dashboard.json` — Paseo Next V2

Import: Grafana → Dashboards → Import → paste JSON → Load → Import. Each file has
a fixed `uid`, so re-importing updates the dashboard in place (choose Overwrite).
