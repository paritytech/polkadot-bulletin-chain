# Bulletin SDK for TypeScript

Off-chain client SDK for Polkadot Bulletin Chain with PAPI integration.

## Installation

```bash
npm install @bulletin/sdk polkadot-api
```

## Build & Test

```bash
npm install
npm run build
npm run test:unit

# Integration tests (requires running node at ws://localhost:9944)
npm run test:integration
```
